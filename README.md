# FitCalory

FitCalory — умный PWA-дневник питания и активности: пользователь вводит еду и тренировки обычным текстом, а приложение примерно считает калории, КБЖУ, расход, дефицит или профицит без backend.

## Возможности

- умный ввод еды: `2 яйца`, `100 г гречки`, `1 куриная грудка`;
- встроенная база продуктов с КБЖУ и примерным весом штуки;
- добавление своих продуктов с сохранением в localStorage;
- умный ввод активности: `бег 1 км`, `зал 45 минут`, `ходьба 3 км`;
- расчёт BMR, базового расхода, дефицита или профицита;
- советы для похудения или набора массы;
- светлая и тёмная тема;
- PWA, manifest и service worker для работы офлайн.

## Запуск

Откройте `index.html` в браузере или разместите файлы на GitHub Pages.
